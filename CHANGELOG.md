# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.3.0] - 2026-01-15

### Changed

- Adopt [pyRTA](https://gitlab.mpi-sws.org/RT-PROOFS/pyRTA) for the computation of response-time bounds. 
- Switch to [pydantic](https://docs.pydantic.dev/latest/) for YAML parsing and validation.
- Refactor virtually all of the codebase, except for the certificate generation code.
- Consistently use type annotations everywhere. 

## [0.2.0] - 2026-01-10

### Changed

- Update POET to work with PROSA v0.6.

## [0.1.5] - 2023-04-01

### Changed

- Update POET to work with PROSA v0.5.

## [0.1.0] - 2022-11-11

Initial public release.
